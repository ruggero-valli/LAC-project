Performs one integration step using the Symplectic Euler method. 

The Symplectic Euler method is a symplectic first order method. It is meant to solve the symplectic cauchy problem \[ \dot{r} = v \dot{v} = f(r,t) \] The integration step of the method is\+: \[ v_{i+1} = v_i + f(r_i, t) dt r_{i+1} = r_i + v_{i+1} dt \] Wikipedia\+: \href{https://en.wikipedia.org/wiki/Semi-implicit_Euler_method}{\texttt{ https\+://en.\+wikipedia.\+org/wiki/\+Semi-\/implicit\+\_\+\+Euler\+\_\+method}}
