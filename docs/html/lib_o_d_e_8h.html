Performs one integration step using the Explicit Euler method.

The Explicit Euler method is the simplest integration method. First order accuracy. The integration step is:

\[ U_{i+1} = U_i + f(U_i, t) dt \]

Wikipedia: https://en.wikipedia.org/wiki/Euler_method

Performs one integration step using the Symplectic Euler method.

The Symplectic Euler method is a symplectic first order method. It is meant to solve the symplectic cauchy problem

\[ \dot{r} = v \dot{v} = f(r,t) \]

The integration step of the method is:

\[ v_{i+1} = v_i + f(r_i, t) dt r_{i+1} = r_i + v_{i+1} dt \]

Wikipedia: https://en.wikipedia.org/wiki/Semi-implicit_Euler_method

Parameters
fis the acceleration
Uis an array that contains the solution vector at time t
tis the time
dtis the integration timestep

If 2N is the number of first order equations in the problem, U is an array of length 2N. The first N elements correspond to the position variables. The elements from N+1 to 2N-1 correspond to the velocity variables, such that for each i, U[N+i] is the velocity associated with the position stored at U[i].
